Formation GOLD | EWAM

Le langage Gold est un langage propriétaire utilisé au sein de l’environnement de développement de Mphasis Wyde.
Il est conçu pour permettre la personnalisation des solutions logicielles, notamment dans le domaine de l’assurance.

Ce langage permet de développer des règles métier, de créer des interfaces utilisateurs dynamiques
et de gérer les interactions entre les différents composants de l’application.

EWAM (Enterprise Web Application Manager) est une API moderne développée par Wyde,
qui facilite la communication entre les composants métiers et les interfaces web.

Elle offre une architecture modulaire et permet une meilleure intégration avec les technologies web actuelles.
