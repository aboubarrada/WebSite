Utilisation de Git et GitHub

Pendant ma première année, j’ai appris à utiliser Git pour versionner mon code localement et suivre mes modifications.

En deuxième année, j’ai utilisé GitHub pour collaborer sur des projets en équipe.
Nous avons travaillé avec des branches, utilisé les pull requests pour intégrer le code après relecture, et géré les tâches via les issues et les projects boards.

Cela a permis une meilleure organisation, une répartition claire du travail et une traçabilité efficace des contributions.
